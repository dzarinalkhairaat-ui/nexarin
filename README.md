# Nexarin by-rins

Nexarin adalah All-in-One Digital Ecosystem / Digital Hub.

## Stack awal

- Next.js
- Tailwind CSS
- Cloudflare Pages
- Static-first
- Mobile-first

## Catatan penting

- Jangan deploy tanpa instruksi.
- Jangan membuat database sebelum frontend public matang.
- Jangan membuat Admin Dashboard sebelum database siap.
- Jangan membuat halaman kosong.
- Halaman yang belum selesai wajib memakai Under Construction.
